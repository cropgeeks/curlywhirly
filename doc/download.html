CurlyWhirly is now available as a beta quality version for initial testing by users. All installers are bundled with a Java 1.6 JRE and Java 3D libraries so there is no need for users to install any other components.

Please uninstall any previous versions of CurlyWhirly before installing a new version.

Choose an installer for the latest version of CurlyWhirly:

Windows Installer
Linux Installer
OSX Installer -- coming soon

Windows installers are tested on Windows XP, Linux installers on Ubuntu release 8, Mac installers on OSX and Intel (and only this combination is supported). Windows and Linux installers may work on other OS versions but no guarantee is given regarding this.

To install, double click the installer icon and follow the instructions.
