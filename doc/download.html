CurlyWhirly is now available as a beta quality version for initial testing by users. All installers are bundled with Java 1.6 JRE and Java 3D libraries so there is no need for users to install any other components.

Choose an installer for the latest version of CurlyWhirly:
Windows CurlyWhirly Windows Installer
Linux CurlyWhirly Linux Installer

Windows installers have been tested with Windows XP but should also work on older versions of Windows.

To install, double click the installer icon and follow the instructions.
