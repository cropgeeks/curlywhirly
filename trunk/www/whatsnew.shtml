Version 1.10.12.10

  • Drag and drop support for file opening
  • Store annotation URLs and automatically fire up web browser with this when data points are clicked
  • Visualization of multi-category datasets
  • Open files from the command line (allows file extension associations)

Version 1.09.05.13

  • Movie recording. Users can now capture a full revolution of the graph about the y axis by using the movie capture option, save the resulting movie as an avi (raw RGB format) file, and then e.g. embed this into a Powerpoint presentation or similar. With your data loaded, just go to File -> Capture movie... and then follow the instructions in the dialog.
